sheaswauger.com — Site Rebuild Spec
Overview
Personal website for Shea Swauger: researcher, writer, and PhD student focused on privacy, data, ethics, and education. Migrating from Wix to a self-hosted static site to eliminate monthly hosting fees.
Stack
	•	Framework: Astro (latest stable)
	•	Styling: Tailwind CSS
	•	Base theme: Astro Sphere (markhorn-dev/astro-sphere)
	•	Language: TypeScript
	•	Hosting: Vercel (free tier)
	•	Domain: sheaswauger.com (currently on Wix — transfer AFTER site is live)
	•	Repo: GitHub (to be created)
Design Decisions
	•	Start from Astro Sphere as base, not from scratch
	•	Remove immediately on scaffold:
	◦	Starfield/stars background component
	◦	Rotating 3D globe component
	◦	Typewriter/cycling text animation
	◦	All "software engineer" / "crypto enthusiast" demo content
	•	Dark mode default (Astro Sphere default), light mode toggle retained
	•	Clean typographic hero: name, tagline, two CTAs
	•	Two-column homepage layout: Recent Posts (left) + Recent Writing (right)
	•	No decorative flourishes — content-forward
Navigation (in order)
	1	Home
	2	Writing
	3	Presentations
	4	Grants
	5	Press
	6	Blog
Room for future additions: Research, CV, Teaching, Newsletter

Presentations (/presentations)
Talks, keynotes, panels. Videos and presentations are unified here — video is an artifact of the presentation, not a separate section.
Each entry has:
	•	Title
	•	Event/venue
	•	Date
	•	Optional: video_url → renders embedded video or play button
	•	Optional: slides_url → renders slides link
	•	Optional: description
If video_url exists, embed renders. If not, entry shows without it. No standalone Videos page.
Content: To be populated from existing Wix presentations page during build.
